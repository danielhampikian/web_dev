HTML Tips and Tricks:

html information page

html defines the structure and content of the webpage

html stands for hypertext markup language
  • html consists of nested elements, where each element consists of
  • A tag that defines the type of element
    1. Such as h for heading
    2. p for paragraph
    3. img for image
    4. a for reference
  • Brackets surrounding the opening and closing of the tag of all nested elements and content in between so that they appear like:
  •  <h1><a>click this link - this is the content between a tags</a></1>
  • Attributes which describe and modify that properties of the tag such as href for so that occur within the first bracket of the tag like so: 
  • <h1><a href="https://examplelink.com">click this link</a></1>
